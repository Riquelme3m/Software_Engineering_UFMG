## Requisitos

- Node.js v22.14

## Como executar

1. Crie um arquivo `.env` na raiz do projeto com o seguinte conteúdo:

```
VITE_ENVIRONMENT=
VITE_API_URL=
```

2. Instale as dependências:

```sh
npm install
```

3. Execute o projeto:

```sh
npm run dev
```

O projeto estará disponível em `http://localhost:5000`
